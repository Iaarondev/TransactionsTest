# Square API Project

This project displays transactions from Square API.

## Setup

1. Install Python requirements:
   ```
   cd backend
   pip install -r requirements.txt
   ```

2. Set your Square API token in `backend/.env`

3. Run the backend:
   ```
   cd backend
   python app.py
   ```

4. Open frontend/index.html in your browser
